Welcome to Reline

Reline is an automatic graph semantic analysis frame with graph editing tools.

Graph semantic analysis Example as follows:

---->A--EdgeX-->B

Reline 使用一种类似antlr的深度遍历方式来遍历给定的图数据

当Reline在遍历图时，下列函数将会被执行: 1.EnterA() 2.EnterEdgeX() 3.EnterB() 4.ExitB() 5.ExitEdgeX() 6.ExitA()

用户在重载上述这些函数(钩子)后,运行Reline就可以操作并分析图。通过这样的方式用户可以轻松制作诸如 将思维导图转化为文档的工具。
